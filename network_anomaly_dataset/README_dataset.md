# UNSW-NB15 derived dataset

* Training source: `UNSW_NB15_training-set.csv`
* Testing source: `UNSW_NB15_testing-set.csv`
* Clients contain only label==0 samples (normal traffic).
* Validation/Test splits keep the original label distribution. Validation/Test/Server sets come from a random split of `UNSW_NB15_testing-set.csv`.
